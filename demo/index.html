Fuzzy Search

Search query is a bit more loose. You can add multiple keywords or characters from a given word/sentence.

query
fuzzy
minChar
1
toHtml
classic
data
dataset1
size
is-small

Custom Html Rendering

You provide your own toHtml render function, in which you can use popular frameworks like bulma, bootstrap, etc.

query
fuzzy
minChar
3
toHtml
custom
data
dataset2
size
is-medium

Custom Query

You can provide a custom query filter function to query through the data you provide. The following exmaple is bananas.

query
custom
minChar
1
toHtml
custom
data
dataset1
size
is-large
